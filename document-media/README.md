## Screenshot Best Practices

1. **Use consistent browser window size** (recommend 1280px width minimum)
2. **Clear browser extensions** or use incognito mode for clean screenshots
3. **Use sample data** that looks realistic and professional
4. **Capture at 2x resolution** for retina displays (scale down in README if needed)
5. **Optimize images** using tools like TinyPNG or ImageOptim before committing
6. **File naming:** Use kebab-case (e.g., `admin-dashboard.png`)

## How to Take Screenshots

### macOS
- **Full screenshot:** `Cmd + Shift + 3`
- **Selection screenshot:** `Cmd + Shift + 4`
- **Window screenshot:** `Cmd + Shift + 4`, then press `Space`

### Windows
- **Full screenshot:** `Win + PrtScn`
- **Selection screenshot:** `Win + Shift + S` (Snipping Tool)

### Browser DevTools
- **Firefox:** `Shift + F2`, then type `screenshot --fullpage`
- **Chrome:** Open DevTools > `Cmd/Ctrl + Shift + P` > "Capture full size screenshot"
